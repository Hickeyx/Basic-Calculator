## Project Title
**Basic-Calculator** main goal for the project is to be able to properly; Add, Subtract, Divide, Multiply and be able to decimals within the mathmatic problem.

## Project Description
**Basic-Calculator** is a simple calculator project made to do basic math problems like adding, subtracting, multiplying, and dividing. It can also work with decimal numbers to give more accurate answers. This project was created to practice basic coding skills like using math in code, handling user input, and making functions work together. One challenge I epxect to face is making sure decimal problems worked correctly and avoiding errors like dividing by zero.
